MockSolo

An app designed to practice drafting a fantasy baseball team. The entire UI was created in code instead of Storyboards. Gets player data from a json file, then uses an algorithm to calculate the next pick based off position in the draft using a snake draft format (1-10, 10-1).

#OptOutside

Created to demo SalesForce machine learning platform, Einstien. The app uses natural language processing to take user input of outdoor interests and how far they are willing to travel. Using this information, the app provides local meetups that meet those parameters.
